## add id 

to add a specific id, place `{: #className}` under it

```md
sentence with id
{: #supId}

```

here `<p id='supId'>sentence with id</p>`


## add class

to add a specific class, place `{: .className}` under it

```md
### title with class
{: .supClass}

```

here `<h3 class='supClass'>title with class</h3>`
